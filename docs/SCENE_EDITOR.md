# 场景编辑器功能说明

## 🎨 新增功能

### 1. 左侧工具栏（对象添加）

```
┌─────────────────┐
│  添加对象       │
├─────────────────┤
│ 🔵 小球         │
│ 📦 箱子         │
│ ━━ 地面         │
├─────────────────┤
│  场景对象       │
│  • 小球 1       │
│  • 箱子 1       │
│  • 地面 1       │
├─────────────────┤
│ [复制] [删除]   │
└─────────────────┘
```

**功能**:
- 点击按钮即可添加新对象到场景
- 自动生成唯一名称（小球 1, 小球 2...）
- 列表显示所有场景对象
- 点击对象可选中并在右侧编辑属性

### 2. 右侧属性面板（实时编辑）

```
┌─────────────────────┐
│  属性编辑器         │
├─────────────────────┤
│  基本               │
│  名称: [小球 1]     │
│  类型: Circle       │
├─────────────────────┤
│  物理               │
│  ☐ 静态对象         │
│  ☑ 使用重力         │
│  质量: 1.00 kg      │
│  恢复系数: 0.80     │
│  摩擦系数: 0.30     │
├─────────────────────┤
│  形状               │
│  半径: 0.50 m       │
├─────────────────────┤
│  变换               │
│  X: 0.00  Y: 0.00   │
├─────────────────────┤
│  外观               │
│  颜色: #3B82F6      │
└─────────────────────┘
```

**支持的属性**:

#### 基本属性
- 名称（可编辑）
- 类型（只读：Circle/Box）

#### 物理属性
- 静态对象（勾选后质量不可编辑）
- 使用重力（是否受重力影响）
- 质量（0.1 - 1000 kg）
- 恢复系数（0 - 1，弹性程度）
- 摩擦系数（0 - 1）

#### 形状属性
- **圆形**：半径（0.1 - 10 m）
- **矩形**：宽度和高度（0.1 - 50 m）

#### 变换属性
- 位置 X/Y（-50 - 50 m）

#### 外观属性
- 颜色（十六进制，如 #3B82F6）

### 3. 项目管理（保存/加载）

**顶部工具栏新增按钮**:
```
📁 新建   💾 保存   📂 打开   ▶ 播放   ⟲ 重置
```

## 📖 使用指南

### 创建场景

1. **添加对象**
   - 点击左侧工具栏的 "🔵 小球" 按钮
   - 对象自动添加到场景中心（0, 0）
   - 在左侧列表中显示

2. **编辑属性**
   - 在左侧列表点击对象
   - 右侧属性面板显示
   - 修改任意属性，实时生效

3. **调整位置**
   - 在属性面板中修改 X/Y 坐标
   - 或者（TODO）直接拖拽画布中的对象

4. **运行仿真**
   - 点击 "▶ 播放" 按钮
   - 观察物体运动
   - 点击 "⏸ 暂停" 停止

### 保存和加载

1. **保存场景**
   ```
   💾 保存 → 自动保存为 "项目名.physicsx"
   ```
   - 默认保存在程序目录
   - JSON 格式，人类可读

2. **加载场景**
   ```
   📂 打开 → 选择 .physicsx 文件
   ```
   - 自动清空当前场景
   - 加载所有对象和属性

3. **新建场景**
   ```
   📁 新建 → 清空所有对象
   ```

### 对象操作

1. **复制对象**
   - 选中对象
   - 点击 "复制" 或按 `Ctrl+D`
   - 副本位置偏移 (1, 1)

2. **删除对象**
   - 选中对象
   - 点击 "删除" 或按 `Delete`

## 🎯 典型工作流

### 示例：创建弹球实验

1. **添加地面**
   ```
   点击 "━━ 地面" → 在属性面板设置：
   - 位置 Y: 8
   - 宽度: 30
   - 高度: 1
   ```

2. **添加小球**
   ```
   点击 "🔵 小球" → 设置属性：
   - 位置 Y: 0
   - 恢复系数: 0.9 (高弹性)
   - 摩擦系数: 0.1 (低摩擦)
   ```

3. **调整重力**
   ```
   顶部工具栏 → 点击 "🌙 月球"
   观察小球在月球重力下的弹跳
   ```

4. **保存场景**
   ```
   💾 保存 → 文件名: "弹球实验_月球.physicsx"
   ```

## 💻 开发者信息

### 数据模型

**SceneProject**:
```csharp
{
    string Name;
    string Version;
    double Gravity;
    List<SceneObject> Objects;
    DateTime CreatedAt;
    DateTime ModifiedAt;
}
```

**SceneObject**:
```csharp
{
    string Id;
    string Name;
    string Type; // "Circle" | "Box"
    Vector2Data Position;
    Vector2Data Velocity;
    double Mass;
    double Restitution;
    double Friction;
    bool IsStatic;
    bool UseGravity;
    float? Radius;
    float? Width;
    float? Height;
    string Color;
}
```

### ViewModel

**SceneEditorViewModel**:
- `ObservableCollection<SceneObjectViewModel>` SceneObjects
- `SceneObjectViewModel?` SelectedObject
- Commands: AddCircle, AddBox, AddGround, Delete, Duplicate, Save, Load

**SceneObjectViewModel**:
- 所有属性都是 ObservableProperty
- `PositionX` / `PositionY` 代理属性（解决 Vector2 绑定问题）
- `ToSceneObject()` / `FromSceneObject()` 转换方法
