PhysicsX 是一个**完整的物理仿真与解题演示系统**，支持：

1. ✅ **精确的物理模拟**（RK4 积分器）
2. ✅ **完整的碰撞系统**（圆形、矩形、摩擦力）

5. ✅ **交互式 UI**（播放/暂停、重置、参数调节）

## 🎮 使用演示

### 基本操作

1. **启动应用**
   ```bash
   dotnet run --project src/PhysicsX.App
   ```

2. **查看默认场景**
   - 两个小球从不同高度落下
   - 绿色地面（静态）
   - 弹性碰撞

3. **播放仿真**
   - 点击 "▶ 播放" 按钮
   - 观察小球自由落体、碰撞、弹起

4. **调整重力**
   - **方法 1**: 在输入框中输入任意值（如 15.5）
   - **方法 2**: 点击预设按钮（如 🌙 切换到月球）
   - 观察小球运动速度变化

5. **重置场景**
   - 点击 "⟲ 重置" 按钮
   - 所有物体回到初始位置
   - 仿真时间归零

## 📐 数学原理

### 1. 自由落体方程

```
位置: y(t) = y₀ + v₀·t + 1/2·g·t²
速度: v(t) = v₀ + g·t
加速度: a = g
```

**验证**: 
- 初始条件: y₀=0, v₀=0
- 1秒后: y = 1/2 × 9.8 × 1² = 4.9 米 ✅

### 2. 碰撞响应（冲量法）

```
冲量大小: j = -(1 + e) · vₙ / (1/m_A + 1/m_B)
速度变化: Δv_A = -j/m_A · n
         Δv_B = +j/m_B · n

其中:
- e: 恢复系数
- vₙ: 法线相对速度
- n: 碰撞法线
```

### 3. 摩擦力（库仑定律）

```
静摩擦: |F_t| ≤ μ_s · |F_n|
动摩擦: F_t = μ_k · |F_n|

其中:
- μ_s, μ_k: 静/动摩擦系数
- F_n: 法向力
```

## 🎨 渲染细节

### Shader Pipeline

```
顶点着色器 (basic.vert)
    ↓
[MVP 变换]
gl_Position = Projection × View × Model × vec4(position, 0, 1)
    ↓
片段着色器 (basic.frag)
    ↓
[颜色输出]
FragColor = vColor
```

### 相机系统

- **投影**: 正交投影（Orthographic）
- **视口**: 自适应宽高比
- **缩放**: 20 像素/米
- **坐标系**: Y 轴向下（屏幕坐标）
